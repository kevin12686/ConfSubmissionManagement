Maintenance

Clear Database and Files

Removes all Paper Master records, Final Submissions, verification states, exceptions, author count rows, settings, conference name, storage previews, system-state backups, uploaded files, reports, CrossCheck exports, and processing outputs. App-owned data folders are staged and removed only after the database transaction succeeds. Configured folders outside the app data/media roots are preserved rather than recursively emptied. After wiping, settings are recreated with defaults so the app can start a completely new conference. Download a System State ZIP from Integrations first if you need to restore this conference later.
